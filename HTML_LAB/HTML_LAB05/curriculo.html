Currículo - Aprendizagens

Pedro Miguel Soares Venda

Ao longo da disciplina de Tecnologias de Internet, fomos desenvolvendo as nossas habilidades no uso de diversas tecnologias e linguagens essenciais para a criação de websites e aplicações web. Entre estas tecnologias, destacam-se o HTML e o XML, fundamentais para a estruturação e apresentação de informações na web.

No HTML, explorámos a criação e manipulação de elementos básicos e avançados. Aprendemos a estruturar um documento HTML utilizando as tags principais, como <body> e <head>. Dentro do <body>, utilizámos diversas tags para formatar e organizar o conteúdo, tais como:

  • Parágrafos: Utilizando a tag <p>, aprendemos a dividir o texto em blocos lógicos e organizados.
  • Marcações: Com a tag <mark>, destacámos partes importantes do texto.
  • Texto Pequeno: A tag <small> permitiu-nos ajustar o tamanho do texto para representar notas de rodapé ou informações menos importantes.
  • Ênfase: Com as tags <em> e <strong>, adicionámos ênfase e destaque ao texto, respectivamente, diferenciando visual e semanticamente as partes importantes.
  • Texto em Negrito e Itálico: Utilizámos <b> e <i> para estilizar o texto, proporcionando variações visuais.
  • Texto Monoespaçado: Com a tag <code>, formatámos trechos de código ou texto que exigem um espaçamento uniforme entre caracteres.
  • Tabelas: Criámos tabelas organizadas e funcionais utilizando <table>, <tr>, <th>, e <td>, facilitando a apresentação de dados tabulares.
  • Listas: Implementámos listas ordenadas (<ol>) e não ordenadas (<ul>) para agrupar e organizar itens de maneira clara e concisa.
  • Imagens: Inserimos e manipulámos imagens no conteúdo utilizando a tag <img>, melhorando a comunicação visual.
  • Âncoras e Links: Com a tag <a>, criámos links navegáveis, facilitando a interconexão entre páginas e secções do site.
  • Linhas Horizontais: Usámos a tag <hr> para criar separações visuais entre secções do conteúdo.
  • Texto Pré-formatado: Utilizámos a tag <pre> para exibir texto exatamente como foi inserido no código, preservando espaços e quebras de linha.

Para além do HTML, também explorámos o XML, uma linguagem de marcação que permite definir regras personalizadas para a estrutura de dados, essencial para a troca e armazenamento de informações de maneira estruturada e legível.

Ao longo do curso, estas ferramentas e técnicas foram aplicadas em diversos projetos práticos, proporcionando uma compreensão sólida de como criar e gerir conteúdo web de maneira eficiente e eficaz. Esta base prepara-nos para enfrentar desafios mais complexos no desenvolvimento web, com uma sólida compreensão dos fundamentos que sustentam a internet.
